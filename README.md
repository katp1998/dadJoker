Hey there!

This is DadJoker - where you can find groan-worthy, cringey and terrorizing dad jokes sourced from https://icanhazdadjoke.com/ 

Start project:
```
composer run setup
composer run dev
```

1. Technologies used:
    - TailwindCSS
    - Eloquent ORM
    - DaisyUI
    - Google Material Icons
    - Google/Bunny fonts
    - Vanilla/AJAX JS
    - Laravel (with SQLite, Vite, composer, NPM)

2. Architecture: MVC laravel architecture
